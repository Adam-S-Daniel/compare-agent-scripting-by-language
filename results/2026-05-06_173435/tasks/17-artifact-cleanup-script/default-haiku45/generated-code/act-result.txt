=== RETENTION POLICY ALGORITHMS ===
The script implements three retention policies that work together:

1. Max Age Policy (days)
   - Deletes artifacts older than the specified age
   - Configuration: max_age_days

2. Keep Latest N Per Run
   - Retains only the latest N artifacts per workflow run ID
   - Deletes older versions within the same run
   - Configuration: keep_latest_n

3. Max Total Size Policy (MB)
   - Enforces a maximum total storage size
   - Deletes oldest artifacts first when limit exceeded
   - Configuration: max_total_size_bytes

Policies are applied in order:
1. First, delete by age
2. Then, enforce keep-latest per run
3. Finally, enforce total size limit
