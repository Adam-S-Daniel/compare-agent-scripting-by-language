### Functions Implemented (TDD order)

1. **`Get-CurrentVersion`** — reads version from `version.txt` (text) or `package.json` (JSON)
2. **`Get-BumpType`** — analyzes conventional commits → `patch`/`minor`/`major`
3. **`Get-NextVersion`** — calculates next semver (patch: x.x.+1, minor: x.+1.0, major: +1.0.0)
4. **`Set-VersionFile`** — writes new version back to the file
5. **`New-ChangelogEntry`** — generates grouped markdown changelog
6. **`Invoke-SemanticVersionBump`** — orchestrates the full pipeline
