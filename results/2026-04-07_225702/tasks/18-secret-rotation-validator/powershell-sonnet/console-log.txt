### Public API (`SecretRotationValidator.ps1`)

| Function | Purpose |
|---|---|
| `Get-SecretStatus` | Classifies one secret: computes `DaysUntilExpiry` and sets `Status` to `expired`/`warning`/`ok` |
| `Invoke-SecretRotationAnalysis` | Processes a collection and groups results into the three urgency buckets with a `Summary` |
| `Format-RotationReport` | Renders a report as `Markdown` (tables with sections) or `JSON`; throws a clear error for unknown formats |
| `New-RotationReport` | End-to-end pipeline: accepts a config hashtable (with string or datetime `LastRotated`), configurable `WarningWindowDays`, and a `Format` choice |

### Classification Logic
- `expired`: `DaysUntilExpiry ≤ 0` (past deadline or exactly today)
- `warning`: `0 < DaysUntilExpiry ≤ WarningWindowDays`
- `ok`: `DaysUntilExpiry > WarningWindowDays`
