- **`Get-SecretStatus`** evaluates a single secret: computes expiry date, days remaining, and urgency (Expired/Warning/OK) based on a configurable warning window
- **`Get-RotationReport`** aggregates all secrets and groups by urgency with summary counts
- **`Format-RotationReport`** outputs either a markdown table (sorted by urgency) or structured JSON
- **`Import-SecretConfig`** loads from JSON files with meaningful error messages for missing/invalid files
- **`ReferenceDate`** parameter enables deterministic testing without mocking `Get-Date`
