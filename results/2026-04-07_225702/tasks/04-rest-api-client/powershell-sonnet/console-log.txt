**Key design decisions:**

- **Mocking:** All HTTP calls are mocked via Pester's `Mock Invoke-RestMethod`. No network calls occur in tests.
- **Cache:** JSON files under a configurable `CacheDir` directory, keyed by `posts_page_N` / `comments_post_N`.
- **Retry:** `Invoke-ApiRequest` retries up to `MaxRetries` times with `BaseDelayMs * 2^attempt` backoff. `BaseDelayMs=0` keeps tests instant.
- **PowerShell array gotcha:** The leading `,` operator (e.g. `return ,$array`) prevents single-element arrays from being silently unrolled at every function return boundary — a non-obvious PowerShell pipeline behavior documented in the file header.
