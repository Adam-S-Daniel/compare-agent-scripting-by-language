### What `New-BuildMatrix` does

1. **Validates** the config is not null, has at least one axis, no empty axes, and valid `max_parallel`
2. **Separates** reserved keys (`fail_fast`, `max_parallel`, `max_size`, `include`, `exclude`) from matrix axes (`os`, `language`, `feature_flags`, etc.)
3. **Computes** cartesian product size (product of all axis value counts) and throws if it exceeds the limit (default 256, or custom `max_size`)
4. **Builds** the output with axes, `fail_fast` (default `$true`), optional `max_parallel`, `include`/`exclude` passthrough, and `matrix_size`
5. **Returns** a JSON string via `ConvertTo-Json -Depth 10`
