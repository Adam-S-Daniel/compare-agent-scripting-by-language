## Key design decisions

- **Glob semantics**: `*` = non-slash chars; `**/` = zero-or-more directory levels; `**` at end = everything
- **Patterns without `/`** are normalized to `**/<pattern>` so `*.test.*` matches test files at any depth
- **Priority** (lower number = higher importance): rules are sorted before evaluation; labels appear in priority order in the output
- **Deduplication**: `HashSet` + `List` combo keeps insertion-order with O(1) duplicate checks
