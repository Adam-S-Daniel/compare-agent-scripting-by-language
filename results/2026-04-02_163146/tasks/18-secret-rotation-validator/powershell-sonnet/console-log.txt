- **`Format-RotationReport`** — renders the report as `JSON` (parseable, nested structure) or `Markdown` (tables with section headers `## Summary`, `## Expired`, `## Warning`, `## Ok`).

- **`Import-SecretConfig`** — loads secret config from a JSON file, validates file existence, handles parse errors, normalises camelCase JSON fields into PascalCase hashtables.
