### Key Design Decisions
- **`IHttpService`** interface → Moq mock replaces real HTTP in all tests
- **`IFileSystem`** interface → `InMemoryFileSystem` (in test file) avoids disk I/O
- **Injectable sleep** in `RetryPolicy` → no-op in tests, `Task.Delay` in production  
- **Cache-aside pattern**: check cache → HTTP on miss → write to cache
- **Pagination**: loops pages until a partial page signals end-of-data
- **Exponential backoff**: delay = `baseDelayMs × 2^attempt` (1s, 2s, 4s, ...)
