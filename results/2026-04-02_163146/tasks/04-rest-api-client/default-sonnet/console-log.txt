### Key design decisions

- **Retry:** `URLError` and HTTP 5xx are retried with `backoff_base * 2^attempt` sleep; HTTP 4xx raises immediately (client error, not transient)
- **Pagination:** fetches pages with `_start`/`_limit` until an empty page or a partial page (< `page_size`)  
- **Cache:** each `(key, value)` pair is a separate `.json` file — persists across process restarts, readable by any Cache instance pointing at the same directory
- **Mocking:** `@patch("client.urllib.request.urlopen")` and `@patch("client.fetch_with_retry")` — no real network calls in any test
