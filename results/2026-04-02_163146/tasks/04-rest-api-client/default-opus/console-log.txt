- **`_http_get_json(url)`** — Low-level HTTP GET using stdlib `urllib` (isolated for mocking)
- **`load_from_cache / save_to_cache`** — JSON file caching with sanitized keys
- **`fetch_paginated(endpoint, ...)`** — Walks all pages using `_start`/`_limit` params
- **`fetch_posts(...)` / `fetch_comments_for_post(post_id, ...)`** — High-level API functions
- **`fetch_posts_with_comments(...)`** — Fetches posts and enriches each with its comments
- Zero external dependencies (stdlib only: `urllib`, `json`, `os`, `time`)

### `test_api_client.py` — 20 Tests in 7 TDD Rounds
| Round | Class | Tests | What it covers |
|-------|-------|-------|----------------|

| 2 | `TestCaching` | 5 | Missing key, roundtrip, dir creation, valid JSON, special chars |
| 3 | `TestPagination` | 5 | Single page, multi-page, empty, URL params, cache integration |
| 4 | `TestFetchPosts` | 2 | Full fetch, retry on error |
| 5 | `TestFetchComments` | 3 | Fetch, URL correctness, cache hit |
| 6 | `TestFetchPostsWithComments` | 2 | Enrichment, network failure |
