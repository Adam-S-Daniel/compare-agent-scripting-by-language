## Key design decisions

- **All interfaces mockable**: `IQueue<T>`, `IItemProcessor<T>`, `IProgressReporter` — no concrete types in pipeline constructor
- **Exponential backoff**: delay = `BaseDelayMs × 2^(retryAttempt-1)`; set `BaseDelayMs = 0` in tests to skip waits
- **Dead-letter queue**: injected via constructor (optional); falls back to internal `InMemoryQueue<T>` if not provided
- **`pipeline.cs`**: all types defined inline for `dotnet run pipeline.cs` self-containment

To run: `dotnet test Pipeline.sln` or `dotnet run pipeline.cs`
