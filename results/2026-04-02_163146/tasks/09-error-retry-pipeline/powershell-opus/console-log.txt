## Files Created

**`ErrorRetryPipeline.ps1`** — Main implementation with 4 functions:

1. **`New-MockQueue`** — FIFO queue backed by `[System.Collections.Generic.Queue[object]]` with closure-based methods (Enqueue, Dequeue, Peek, Count, IsEmpty, GetAll)
2. **`Get-ExponentialBackoffDelay`** — Calculates `min(baseDelayMs * 2^attempt, maxDelayMs)`
3. **`Invoke-WithRetry`** — Executes a scriptblock with configurable retry + exponential backoff. Returns `{Success, Result, Error, Attempts}`. `SleepAction` parameter makes sleep mockable.
4. **`Invoke-Pipeline`** — Processes all items from a source queue, retrying failures. Failed items go to a dead-letter queue. Reports progress via `OnProgress` callback. Returns a final summary with `{TotalItems, Processed, Failed, Retried, DeadLetterQueue, Results}`.
