- **`dirsync.py`** — Main module with 6 functions built through TDD cycles:
  1. `hash_file()` — SHA-256 hex digest of file contents (chunked reads for large files)
  2. `scan_tree()` — Walk a directory, return `{relative_path: sha256_hash}` dict
  3. `compare_trees()` — Compare two trees, classify files into `only_in_source`, `only_in_dest`, `content_differs`, `identical`
  4. `generate_sync_plan()` — Produce list of COPY/UPDATE/DELETE actions
  5. `dry_run()` — Human-readable report (no changes made)
  6. `execute_sync()` — Perform the actual sync operations
