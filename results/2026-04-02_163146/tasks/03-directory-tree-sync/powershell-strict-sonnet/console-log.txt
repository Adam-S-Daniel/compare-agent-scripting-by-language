## Solution Overview

Two files created in the working directory:

### `DirectorySync.Tests.ps1` — Tests (written first, TDD red phase)
**30 tests** across 6 `Describe` blocks covering every function:

| Describe | Tests |
|---|---|
| `Get-FileHash256` | hash length, different content → different hash, same content → same hash, throws on missing file |
| `Get-DirectoryIndex` | returns hashtable with relative paths, stores 64-char hashes, empty dir → empty table, throws on missing dir |
| `Compare-DirectoryTrees` | identifies Identical/Modified/OnlyInSource/OnlyInTarget, works with nested subdirs |
| `New-SyncPlan` | CopyToTarget for source-only, UpdateInTarget for modified, DeleteFromTarget for target-only, no actions for identical |
| `Invoke-SyncPlan (dry-run)` | does NOT modify files, returns report of planned actions |
| `Invoke-SyncPlan (execute)` | copies new files, updates modified files, deletes orphans, creates subdirs, returns success report |
| `Invoke-DirectorySync` | end-to-end dry-run leaves files unchanged, end-to-end execute fully syncs target to source |

### `DirectorySync.ps1` — Implementation (green phase)

**Strict mode compliance throughout:**
- `Set-StrictMode -Latest` + `$ErrorActionPreference = 'Stop'` at top
- Every function has `[CmdletBinding()]`, `[OutputType()]`, and explicitly typed parameters
- No implicit type conversions — explicit casts used everywhere
- All variables declared with type annotations

**Functions:**
1. **`Get-FileHash256`** — SHA-256 hash via `Get-FileHash`, returns lowercase 64-char hex string
2. **`Get-DirectoryIndex`** — Recursive scan → relative-path-to-hash `[hashtable]`
3. **`Compare-DirectoryTrees`** — Diffs two indexes → `PSCustomObject` with `Identical`/`Modified`/`OnlyInSource`/`OnlyInTarget` arrays
4. **`New-SyncPlan`** — Converts comparison result → `PSCustomObject[]` of `{Action, RelativePath}` items
5. **`Invoke-SyncPlan`** — Executes or dry-runs a plan, returns result report with `{Action, RelativePath, Status, Error}`
6. **`Invoke-DirectorySync`** — Top-level orchestrator combining all of the above
